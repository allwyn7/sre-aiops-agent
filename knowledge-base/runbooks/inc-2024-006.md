**Trigger Pattern:** `CrashLoopBackOff with 'Reason: OOMKilled' and high resource usage`

## 1. Detection

Monitor the following for early detection: 
  - Log Events: 'OOMKilled: container <container-name> in pod <pod-name>'
  - Metrics: 'container_memory_usage_bytes', 'container_memory_working_set_bytes', container memory limit breaches, high error rates (e.g., > 5%), and increased HTTP 5xx response rate.
